HTML Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

1. HTML Style Attribute

The HTML style attribute has the following syntax:

<tagname style="property:value;">

You have to add style attribute in the tag

1.1 Background Color

The CSS background-color property defines the background color for an HTML element:

<body/div/h1/p style="background-color:powderblue;">

This changes the background-color.

if you want the whole page to have a background color, you can apply it to the <body> element.

1.2 Text Color

The CSS color property defines the text color for an HTML element:

<body/div/h1/p style="color:blue;">

This changes the text color.

1.3 Font

The CSS font-family property defines the font for an HTML element:

<body/div/h1/p style="font-family:Arial, sans-serif;">

You can change fonts.

1.4 Font Size

The CSS font-size property defines the font size for an HTML element:

<body/div/h1/p style="font-size:20px;">

You can change font size.

1.5 Text Alignment

The CSS text-align property defines the horizontal alignment for an HTML element:

<body/div/h1/p style="text-align:center;">

You can change text alignment.

1.6 Multiple Styles

You can also add multiple styles to an element by separating each property with a semicolon (;):

<body/div/h1/p style="property1:value1; property2:value2; ...">

This paragraph has multiple styles applied.
