HTML Text Formatting

HTML contains several elements for defining text with a special meaning.

HTML Formatting Elements

The following table lists some commonly used HTML formatting elements:

        <b> - Bold text
        <strong> - Important text
        <i> - Italic text
        <em> - Emphasized text
        <mark> - Marked text
        <small> - Smaller text
        <del> - Deleted text
        <ins> - Inserted text
        <sub> - Subscript text
        <sup> - Superscript text

1. HTML <b> and <strong> Elements

The HTML <b> element defines bold text, without any extra importance.

This text is bold

The HTML <strong> element defines important text.

This text is important

<b> shows how text looks, but <strong> tells the browser and assistive tools that the text is important, even though they look the same.

2. HTML <i> and <em> Elements

The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

This text is italic

The HTML <em> element defines emphasized text. The content inside is typically displayed in italic.

This text is emphasized

<em> tells the browser and screen readers that the text is stressed or emphasized, while <i> is just how it looks.

3. HTML <mark> Element

The HTML <mark> element defines marked or highlighted text.

This text is marked or highlighted

4. HTML <small> Element

The HTML <small> element defines smaller text.

This text is smaller

5. HTML <del> Element

The HTML <del> element defines deleted text.Browsers will usually strike a line through deleted text:

This text is deleted

6. HTML <ins> Element

The HTML <ins> element defines inserted text. Browsers will usually underline inserted text:

This text is inserted

7. HTML <sub> Element

The HTML <sub> element defines subscript text.

This is subscripted text.

8. HTML <sup> Element

The HTML <sup> element defines superscript text.
