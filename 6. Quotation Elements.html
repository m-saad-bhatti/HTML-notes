HTML Quotation and Citation Elements

1. The HTML <q> — Inline Quotation Element

The HTML <q> element is used to define a short inline quotation. Browsers typically render the text within this element with quotation marks.

For Example: Einstein said, Imagination is more important than knowledge.

Syntax: 

<q>...</q>

        Key Points
            1. Browser automatically adds quotation marks
            2. Best for short quotes
            3. Inline element (does not break lines)

2. The HTML <blockquote> — Block Quotation Element

The HTML <blockquote> element is used to define a long block quotation. Browsers typically render the text within this element with indentation.

For Example:

Education is the most powerful weapon which you can use to change the world.

Syntax: 

<blockquote>...</blockquote>

2.1 "cite" attribute in <blockquote>

The <blockquote> element can also include a "cite" attribute to specify the source of the quotation.

For Example:

The only limit to our realization of tomorrow is our doubts of today.

Syntax: 

<blockquote cite="URL">...</blockquote>

        Notes:
            The URL is not visible to users
            Useful for search engines and tools
        Key Points
            1. Browser indents the quotation
            2. Best for long quotes
            3. Block-level element (breaks lines)
            4. "cite" attribute specifies the source URL

3. The HTML <cite> — Citation Element

The HTML <cite> element is used to define the title of a work, such as a book, article, or song. Browsers typically render the text within this element in italics.

For Example: My favorite book is Atomic Habits by F. Scott Fitzgerald.

Syntax: 

<cite>...</cite>

        Key Points
            1. Browser typically italicizes the text
            2. Represents the source work
            3. Inline element (does not break lines)

4. The HTML <abbr> — Abbreviation Element

The HTML <abbr> element is used to define an abbreviation or acronym. Browsers typically display the text in italics and show a tooltip with the full form when hovering over the abbreviation.

For Example: The WHO is a global health organization.

Syntax: 

<abbr title="Full Form">Abbreviation</abbr>

        Key Points
            1. Browser typically italicizes the text
            2. "title" attribute provides full form on hover
            3. Inline element (does not break lines)

5. The HTML <address> — Contact Information Element

The

HTML <address> element
is used to define the contact information of the author or owner of a document or article. Browsers typically render the text within this element in italics.

For Example:

John Doe
123 Main Street
Anytown, USA

Syntax: 

<address>...</address>

        Key Points
            1. Browser typically italicizes the text
            2. Used for contact information
            3. Block-level element (breaks lines)

6. The HTML <bdo> — Bidirectional Text Element

The HTML <bdo> element is used to override the default text direction. It allows you to change the direction of text within an element, such as from left-to-right (LTR) to right-to-left (RTL).

For Example: This text will be displayed in right-to-left direction.

Syntax: 

<bdo dir="ltr|rtl">...</bdo>

        Key Points
            1. "dir" attribute specifies the text direction
            2. "ltr" for left-to-right, "rtl" for right-to-left
            3. Inline element (does not break lines)

Chapter Summary

        1. Use <q> for short inline quotations
        2. Use <blockquote> for long block quotations
        3. Use <cite> for titles of works
        4. Use <abbr> for abbreviations and acronyms
        5. Use <address> for contact information
        6. Use <bdo> to override text direction
