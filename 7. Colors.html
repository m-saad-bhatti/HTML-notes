HTML Colors

HTML colors are used to style text, backgrounds, borders, and other elements. Colors can be defined using different color models such as RGB, HEX, and HSL.

1. RGB Colors

RGB stands for Red, Green, Blue. Each value ranges from 0 to 255.

rgb(255, 0, 0) → Red

rgb(0, 255, 0) → Green

rgb(0, 0, 255) → Blue

Mixing values creates new colors.

rgb(255, 165, 0) → Orange

Special RGB values:

  • rgb(0, 0, 0) → Black
  • rgb(255, 255, 255) → White
  • rgb(128, 128, 128) → Gray

2. HEX Colors

HEX colors use hexadecimal values. They start with # followed by 6 characters.

Format: #RRGGBB

#FF0000 → Red

#00FF00 → Green

#0000FF → Blue

HEX shorthand exists when both digits are the same.

#000 → Black

#FFF → White

3. HSL Colors

HSL stands for Hue, Saturation, Lightness. This model is more human-friendly.

Syntax: hsl(hue, saturation%, lightness%)

  • Hue: Color angle (0–360)
  • Saturation: Color intensity
  • Lightness: Brightness

hsl(0, 100%, 50%) → Red

hsl(120, 100%, 50%) → Green

hsl(240, 100%, 50%) → Blue

Adjusting lightness:

Dark Blue (30% lightness)

Light Blue (70% lightness)

Chapter Summary

  • RGB uses values from 0 to 255
  • HEX is a hexadecimal representation of RGB
  • HSL is easier to adjust for brightness and intensity
  • All models represent the same colors differently
