HTML + CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance of a webpage. HTML creates the structure, CSS controls how it looks.

With CSS, you can change colors, fonts, text size, spacing, layout, and how content appears on different screen sizes.

What does "Cascading" mean?

Cascading means styles flow from parent elements to child elements. If a style is applied to the body, it affects all text inside it unless another style overrides it.

1. Inline CSS

Inline CSS is written directly inside an HTML element using the style attribute.

This heading uses inline CSS

syntax: <tagname style="property:value;"> content </tagname>
This paragraph uses inline CSS.

Inline CSS should be used only for testing or single elements. Using it everywhere makes code hard to manage.

2. Internal CSS

Internal CSS is written inside a style tag in the head section of the HTML document.
syntax: <style> CSS rules </style>

It is useful when one page needs multiple styles. The styles at the top of this file are an example of internal CSS.

3. External CSS

External CSS is written in a separate .css file. It is linked to the HTML document using the <link> tag. This method is best for reusing styles across multiple pages.
syntax: <link rel="stylesheet" href="styles.css">

Chapter Summary

HTML defines structure. CSS defines appearance. Inline CSS styles single elements. Internal CSS styles an entire page. Border, padding, and margin control spacing.
