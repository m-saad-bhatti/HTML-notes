# HTML Notes

My learning notes and practice while studying **HTML**.  
This repository contains basic concepts, explanations, and examples that I covered while learning HTML fundamentals.

---

## Topics Covered

### 1. HTML Attributes
Examples and usage of common HTML attributes, including:

- **href** attribute  
- **src** attribute  
- **style** attribute  
- **lang** attribute  
- **title** attribute  

These attributes are used to provide additional information about HTML elements and control their behavior or appearance.

---

### 2. HTML Headings
HTML headings are titles or subtitles displayed on a webpage.

- Defined using `<h1>` to `<h6>` tags  
- `<h1>` is the most important heading  
- `<h6>` is the least important heading  
- Headings help structure content  
- Search engines use headings to understand page structure  

Includes examples showing heading hierarchy and usage.

---

### 3. HTML Paragraphs
HTML paragraphs are defined using the `<p>` tag.

Key concepts covered:

- Default paragraph behavior  
- How browsers ignore extra spaces and lines  
- Line breaks using the `<br>` element  
- Horizontal rules using the `<hr>` element  
- Preformatted text using the `<pre>` element  

Examples demonstrate how text is rendered differently depending on the element used.

---

### 4. HTML Styles
The `style` attribute is used to apply CSS directly to HTML elements.

Topics include:

- **Background color** (`background-color`)  
- **Text color** (`color`)  
- **Font family** (`font-family`)  
- **Font size** (`font-size`)  
- **Text alignment** (`text-align`)  
- Applying multiple styles in one element  

This section explains inline styling and its syntax.

---

### 5. HTML Text Formatting
HTML provides elements to format text and give it semantic meaning.

Formatting elements covered:

- `<b>` – Bold text  
- `<strong>` – Important text  
- `<i>` – Italic text  
- `<em>` – Emphasized text  
- `<mark>` – Highlighted text  
- `<small>` – Smaller text  
- `<del>` – Deleted text  
- `<ins>` – Inserted text  
- `<sub>` – Subscript text  
- `<sup>` – Superscript text  

Differences between visual formatting and semantic meaning are explained with examples.

---

### 6. HTML Quotation and Citation Elements
Elements used for quotations, references, and text direction.

Covered elements:

- `<q>` – Inline quotations  
- `<blockquote>` – Block quotations  
- `cite` attribute in `<blockquote>`  
- `<cite>` – Title of a work  
- `<abbr>` – Abbreviations and acronyms  
- `<address>` – Contact information  
- `<bdo>` – Bidirectional text override  

Each element includes explanation, syntax, and key points.
